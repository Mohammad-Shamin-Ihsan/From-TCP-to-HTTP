# TCP to HTTP (Go)

A project built while learning Go and networking fundamentals.

## Concepts Covered

- Go modules
- File I/O
- Byte buffers
- Slices
- Interfaces (`io.ReadCloser`)
- Goroutines
- Channels
- TCP servers
- Reading streamed data
- Parsing complete lines from byte streams

## Run

```bash
go run .
```

The server listens on port `42069`.
